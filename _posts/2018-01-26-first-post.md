기존에는 노션을 블로그처럼 사용했었는데,
다시 GitHub Pages 기반의 블로그로 돌아오기로 했다.

노션도 충분히 좋은 환경이지만, 바꾼 이유는 다음과 같다.
1. 글이 늘어날 수록 이유모를 불편함이 느껴진다.
2. '문서처럼 써야될 것 같은 부담감' 이 생긴다.
3. 커맨드들의 조합이 조금 불편하다. (cmd + shift + 화살표 등)
4. 유료 서비스가 아니면, 검색 엔진 노출이 되지 않는다.
5. 구글 애널리틱스를 활용해보고 싶어졌다.
6. GitHub 서비스에 문제가 생기지 않는다면, 안정적이다.
   (참고로, 노션으로 넘어간 이유가 'GitHub 서버가 터져서' 였다.)

To add new posts, simply add a file in the `_posts` directory that follows the convention `YYYY-MM-DD-name-of-post.ext` and includes the necessary front matter. Take a look at the source for this post to get an idea about how it works.
