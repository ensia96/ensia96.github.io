# 컴퓨터 구조

## Microprocessor Cache

```
┌─────┐    address     ┌─────── Cache ───────┐        ┌────────────┐  address   ┌────────┐
|     | ─────────────→ |                     |  miss  |            | ─────────→ |        |
| MPU |                | - Cache Memory      | ─────→ | Memory Bus |            | Memory |
|     | ←─── data ───→ | - Cache Controller  |        |            | ←─ data ─→ |        |
└─────┘   (cache hit)  |                     |        └────────────┘            └────────┘
  ↑                    └─────────────────────┘              ↑
  |                               ↑                         |
  |                               | update                  |
  |                               |                         |
  └───────────────────────────────┴─────────────────────────┘
                             data
```

- 여기서 MPU는 'Micro Processing Unit' 의 약자로, 마이크로프로세서라고 부르기도 한다.
- 이러한 MPU는 하나의 집적 회로로 구성되어 있으며 주어진 연산을 처리하는 역할을 한다.

```
          | MPU |
          └─────┘
             |
             | virtual address
             ↓
          ┌─────┐
          | MMU |
          └─────┘
             |
             | physical address
             ↓
    ┌ Physical Memory ┐
    |─────────────────|
┌──→|        1        |
|   |─────────────────|
|   |        2        |←┐
|   |─────────────────| |
| ┌→|        3        | |
| | └─────────────────┘ |

└─┼─|        1        | |
  | |─────────────────| |
  └─|        3        | |
    |─────────────────| |
    |        2        |─┘
    └─────────────────┘
             ↑
         ┌──────┐
         | Disk |
         └──────┘
```
