# 컴퓨터 구조

## Microprocessor Cache

```
┌─────┐    address     ┌─────── Cache ───────┐        ┌────────────┐  address   ┌────────┐
|     | ─────────────→ |                     |  miss  |            | ─────────→ |        |
| MPU |                | - Cache Memory      | ─────→ | Memory Bus |            | Memory |
|     | ←─── data ───→ | - Cache Controller  |        |            | ←─ data ─→ |        |
└─────┘   (cache hit)  |                     |        └────────────┘            └────────┘
  ↑                    └─────────────────────┘              ↑
  |                               ↑                         |
  |                               | update                  |
  |                               |                         |
  └───────────────────────────────┴─────────────────────────┘
                             data
```

- 여기서 MPU는 'Micro Processing Unit' 의 약자로, 마이크로프로세서라고 부르기도 한다.
- 이러한 MPU는 하나의 집적 회로로 구성되어 있으며 주어진 연산을 처리하는 역할을 한다.

## Virtual Memory(Memory Management)

```
          ┌─────┐
          | MPU |
          └─────┘
             |
             | virtual address
             ↓
          ┌─────┐
          | MMU |
          └─────┘
             |
             | physical address
             ↓
    ┌ Physical Memory ┐
    |─────────────────|
┌──→|        1        |
|   |─────────────────|
|   |        2        |←┐
|   |─────────────────| |
| ┌→|        3        | |
| | └─────────────────┘ |
| | ┌ Virtual Memory ─┐ |
└─┼─|        1        | |
  | |─────────────────| |
  └─|        3        | |
    |─────────────────| |
    |        2        |─┘
    └─────────────────┘
             ↑
         ┌──────┐
         | Disk |
         └──────┘
```

- 여기서 MMU는 'Memory Management Unit' 의 약자이며, 메모리 관리 장치라고 부른다.
- 메모리 관리 장치의 주된 역할은 가상 메모리 주소를 물리 메모리 주소로 변환하는 것이다.

## How to translate Virtual Adress to Physical Address!

```
VPN = Virtual Page Number
PPN = Physical Page Number
PO = Page Offset

┌─────┬────┐
| VPN | PO |
└─────┴────┘
   |
   | Translate
   ↓
┌─────┬────┐
| PPN | PO |
└─────┴────┘
```

### 1. Basic Translation

```
┌─────┬────┐
| VPN | PO |
└─────┴────┘
   |        Page Table
   |   ┌──┬───────┬───────┐
   |   |  | VPN 3 | PPN 1 |
   |   |──┼───────┼───────|    ┌─────┬────┐
   └─→ |  | VPN 1 | PPN 2 | ─→ | PPN | PO |
       |──┼───────┼───────|    └─────┴────┘
       |  | VPN 2 | PPN 3 |
       └──┴───────┴───────┘
```

### 2. Two-level Paging

```
   Page Directory

|  |       |       | ─→ |     |    |
|──┼───────┼───────|    └─────┴────┘
|  |       |       |
└──┴───────┴───────┘
```
