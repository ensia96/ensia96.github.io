## (1) TCP 통신과 UDP 통신의 차이점:

TCP 는 통신 정확도를 우선시하는 방식의 통신 규약입니다.

UDP 는 통신 속도를 우선시하는 방식의 통신 규약입니다.

TCP 의 경우, 패킷 전송 성공 여부를 매번 확인하기 때문에 신뢰도는 높지만 그만큼 느립니다.

반대로 UDP 의 경우, 패킷 전송 성공 여부보다는 속도를 우선시하기 때문에 훨씬 더 빠릅니다.


## (2) TCP 통신을 사용하는 곳:

TCP 통신은 정보가 온전히 전달되는 것이 중요한 상황에서 주로 사용됩니다.

이메일 서비스, 파일 전송 서비스 등을 예로 들 수 있습니다.


## (3) UDP 통신을 사용하는 곳:

UDP 통신은 정보 전달의 정확성보다 통신 속도가 더 중요한 상황에서 주로 사용됩니다.

영상 통화 서비스, 실시간 스트리밍 서비스 등을 예로 들 수 있습니다.
