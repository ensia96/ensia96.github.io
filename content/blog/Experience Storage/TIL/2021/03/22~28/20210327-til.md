# 오늘 한 일

- Crash Course 내용 정리
   - '10' 의 내용 정리를 마무리했다.
   - '11' 의 내용을 모두 받아 적었다.
- 천공 카드 판독기에 대해 공부했다.
   - 천공 카드에 저장된 정보를 컴퓨터에 입력하는데 사용되었다.
   - 한 장씩 천공 카드를 빨아들인다.
   - 카드의 내용을 컴퓨터 기억 장치에 써넣는다.
   - 여러 장을 입력받으면, 여러 정보를 블록으로 저장한다.
   - 프로그래밍 정보가 입력된 경우, 실행하라는 표시를 출력한다.
- 프로그램의 순서를 기록하는데 사용된 스트라이핑이라는 기법에 대해 알게됐다.  
  `(카드 뭉치 옆면에 대각선으로 선을 긋는 방식이다.)`
- 천공 카드를 이용해 컴퓨터에서 정보를 출력하는 것도 가능하다는 것을 알게됐다.
- 패널 프로그래밍에 대해 알게됐다.
   - 큰 판에 달린 버튼이나 스위치를 이용하여 컴퓨터를 제어한다.
   - 기억 장치 내부에 있는 값이나 기능을 표시하는 표시등이 있다.
- Altair 8800 에 대해 알게됐다.
   - 최초의 가정용 컴퓨터, 상업적으로 매우 성공했다.
   - 완성형과 조립형, 2가지 형태로 출시되었다.
   - 조립형의 경우 매우 저렴해서 매니아들 사이에서 인기가 많았다.
   - 앞면에 여러 개의 스위치와 표시등이 있다.
   - '명령 코드 입력, 넣어두기' 를 반복하며 프로그래밍한다.
