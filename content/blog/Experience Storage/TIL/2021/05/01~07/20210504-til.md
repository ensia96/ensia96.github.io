# 오늘 한 일

- 자료 구조, 알고리즘 수업 정리
   - '자료구조 스택 (stack)' 이라는 강의 내용을 정리했다.
- 스택에 대해 공부했다.
   - 순차적 자료 구조 중 하나이며, 최소 2개의 연산(함수) 을 제공한다.
      - 삽입 연산을 push(), 삭제 연산을 pop() 이라 한다.
   - 리스트를 수직으로 세워놓은 듯한 형태로 표현할 수 있다.
   - 값은 차례대로 삽입되며, 중간에 값을 끼워 넣을 수 없다.
   - 값이 제거될 때는 반대 순서로, 마지막에 들어온 값이 하나씩 제거된다.
   - 리스트의 다른 연산을 혼용할 수 있어서, 별도의 클래스를 구성하는 것이 좋다.
   - 마지막 값을 반환하는 top()과 크기를 반환하는 len() 함수를 제공할 수 있다.
   - push(), pop(), top(), len() 의 연산 수행 시간은 모두 상수 시간 O(1) 이다.
   - 따라서, 스택은 모든 연산이 상수 시간에 수행되는 자료 구조라고 할 수 있다.
   - 괄호 맞추기와 같은 유형의 알고리즘에서 활용하기 좋은 자료 구조다.
- Python 의 클래스에 대해 공부했다.
   - class 키워드로 선언을 시작하며, 클래스 이름은 관례상 대문자로 시작한다.
   - 매직 메소드(special method) 는 이름이 정해져 있는 특별한 함수다.
   - '\__init\__' 이라는 매직 메소드는 생성 함수이며, 이것이 실행되면 객체가 생성된다.
      - 클래스 이름을 함수처럼 호출하면 자동으로 생성 함수가 호출되며, 객체가 반환된다.
      - 생성 함수가 없으면, 아무런 멤버 변수도 갖지 않는 기본 생성 함수가 호출된다.
   - 생성 함수의 매개 변수는 해당 클래스의 객체를 가리키는 self 로 시작한다.
      - 다른 함수를 지정할 때도 마찬가지이며, 이외의 매개 변수는 원하는 대로 추가할 수 있다.
   - 생성 함수에서 다양한 멤버 변수들을 선언할 수 있다.
      - 이러한 멤버 변수에는 self 에 멤버를 나타내는 연산자 '.' 을 붙여 접근할 수 있다.
   - 해당 클래스의 객체에 대해 len() 함수를 이용하려면, '\__len\__' 라는 매직 메소드를 선언해야 한다.
      - len() 함수가 호출되면, 파이썬은 해당 객체 내부의 '\__len\__' 를 호출한다.
- Python 의 예외 처리에 대해 공부했다.
   - 예외(exception) 혹은 에러 처리(error handling) 는 try except 구문으로 수행할 수 있다.
   - try 블록에 있는 문장들을 말 그대로 시도(try) 하여, 에러가 있는지 확인할 수 있다.
   - except 키워드 뒤에 명시된 에러가 발생하면, 해당 except 블록의 문장들이 실행된다.
      - 예를 들어, IndexError 는 유효한 인덱스를 명시하지 않았을 때 발생한다.
