# 오늘 한 일

- Crash Course 내용 정리
   - '6. 벡터 그래픽' 의 초반부까지 정리했다.
- 음극선관에 대해 공부했다.
   - 인광체에 전자가 닿으면 빛이 나는 원리를 이용하는, 초기의 영향력 있는 디스플레이 기술이다.
   - 방사체에서 전자를 발사하며, 금속판이나 코일(전자기장) 을 이용해 전자의 경로를 조작한다.
   - 그래픽을 그리기 위해 빛을 주사하는 방법으로 벡터 주사, 래스터 주사 방식을 사용한다.
- 벡터와 래스터에 대해 공부했다.
   - 벡터와 래스터는 여러 그래픽 요소를 화면에 표시하는 그려내는 방법이다.
   - 벡터는 인광체의 빛이 유지되는 동안, 형체를 추적하여 빛을 반복적으로 주사하는 방법이다.
   - 래스터는 왼쪽 위에서부터 오른쪽 아래로 한 줄씩, 특정 지점에만 빛을 주사하는 방법이다.
- 액정표시장치에 대해 공부했다.
   - 래스터 주사 방식을 사용하여 작은 화소의 밝기를 초당 여러 번 갱신하는 장치다.
   - 화소 정보는 메모리를 너무 많이 사용해서, 초기 컴퓨터에서는 사용되지 못했다.
- 문자 발생기에 대해 공부했다.
   - 문자 격자를 사용하는 데에 필요한 하드웨어이며, 최초의 그래픽 카드다.
   - RAM에서 읽어온 문자를 래스터 그래픽으로 변환하여 화면에 그려내는 장치다.
   - 내부에는 문자를 나타내는 그래픽 요소가 저장된 읽기 전용 기억 장치가 있다.
      - 이 때, 이러한 그래픽 요소를 점 행렬이라고 한다.
   - 그래픽을 위해 예약된 메모리 영역인 화면 버퍼에 대한 특별한 접근 권한이 있다.
   - 화면 버퍼에 저장된 값들을 조작함으로써, 여러 문자를 화면에 렌더링한다.
- 아스키 아트에 대해 공부했다.
   - 문자 집합에 속한 기호로 만들어진 여러 모양의 그림 정보다.
   - 정교한 작업을 처리하기 위해 다양한 확장 집합이 만들어졌다.
   - 추가 비트로 문자 색상과 배경색을 정의할 수 있는 체계도 있다.
   - DOS 운영 체제의 그래픽 인터페이스는 아스키 아트로 구성되었다.
