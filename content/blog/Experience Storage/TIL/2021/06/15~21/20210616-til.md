# 오늘 한 일

- 자료 구조, 알고리즘 수업 정리
   - '알고리즘 분할정복법 소개 - 피보나치 수' 라는 강의 내용을 정리했다.
- 행렬 곱셈을 이용해 피나보치 수를 구하는 방법에 대해 공부했다.
   - n번째 피보나치 수를 Fn 이라고 했을 때, F0 = 0, F1 = 1, Fn = F(n - 1) + F(n - 2) 다.
   - 이 때, (Fn, F(n - 1)), (1, 1, 1, 0), (F(n - 1), F(n - 2)) 의 행렬을 각각 N, A, B라고 한다.
   - 그리고, 행렬 N, A, B에 대한 등식, N = A * B 가 성립한다는 것을 보여, 문제를 해결한다.
   - B는 N에서 n이 1씩 줄어든 것과 같으므로, B = A * (F(n - 2), F(n - 3)) 라고 할 수 있다.
   - 따라서, N = A * B = A * (A * (F(n - 2), F(n - 3))) = A^2 * (F(n - 2), F(n - 3)) 가 된다.
   - 이렇게, 행렬식을 계속 전개하면, N = A^(n - 1) * (F1, F0) = A^(n - 1) * (1, 0) 이 된다.
   - 전개한 행렬식에서 n을 1만큼 증가시키면, (F(n + 1), Fn) = A^n * (1, 0) 의 형태가 된다.
   - 행렬식을 풀기 위해, 분할 정복법을 이용하는 거듭제곱 알고리즘으로 A^n 을 구할 수 있다.
   - 마지막으로, A^n 에 (1, 0) 행렬을 곱하면, n, (n + 1) 번째 피보나치 수를 구할 수 있다.
   - 이는, 거듭제곱 알고리즘의 수행 시간, O(log n) 만큼의 수행 시간이 필요한 알고리즘이다.
