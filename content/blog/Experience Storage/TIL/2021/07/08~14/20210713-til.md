## 생각 정리

코딩으로 힐링을 해서 기분이 좋아졌다.. Crash Course 정리를 다시 시작해도 될 것 같다.

- 전처럼, 스트레스가 쌓이지 않도록, 공부는 꾸준히 하되, 중간중간 코딩도 하기로 했다.
- 공부하는 것에 스트레스를 받거나 부담을 느끼지 않도록 하는 방법도 찾아봐야겠다. ㅎ

<br>

페이지를 이동하거나 다른 곳을 클릭하면 사이드바의 폴더들이 닫히는 것이 좀 불편하다.

- 폴더의 열린 상태를 유지하도록 코드를 개선하고, 모두 닫기 기능도 추가해야겠다.
- 세션 스토리지를 이용해서, '열림' 상태를 기록하도록 구성하면 가능할 것 같다.
- \<details> 의 open 상태를 지정하는 방법에 대해서는 조금 알아봐야 할 것 같다.
- 블로그가 넓은 화면에 렌더링될 때, 컨텐츠 영역을 좁히면 좋을 것 같다는 생각이 들었다.
- 블로그에 태그별로 필터링하는 기능을 추가하면, 더 좋을 것 같다는 생각도 들었다.
