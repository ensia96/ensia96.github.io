오늘 알고리즘 문제는 얼마 전에 유튜브에서 본 적이 있어서, 그리 어렵지 않게 풀 수 있었다.

- 대신에, 그때 봤던 풀이대로 구현하기는 싫어서, 코드 구성을 약간 다르게 해서 풀었다.
- 미세먼지 확산은 쉽게 구현했는데, 공기 청정기 동작은 구현하는 데에 조금 오래 걸렸다.

<br>

'코드 주입 공격 - 배경지식' 에 대해 공부했다.

- 코드 주입이라는 기법을 이용하는 공격도 취약점 공격의 전형적인 유형 중 하나이다.
- 코드 주입은 데이터베이스 기반 웹 사이트를 공격할 때 가장 자주 사용되는 기법이다.
- 데이터베이스를 대상으로 한 코드 주입 공격은 구조화된 질의 언어를 통해 수행된다.
- 서버가 데이터베이스에서 정보를 조회하는 데 사용하는 명령어를 SQL 질의라고 한다.
- 정보를 저장하는 데에 테이블이라는 자료 구조를 사용하는 데이터베이스 체계가 있다.
- SQL 질의는 조회할 정보, 조회할 위치, 조건 등으로 구성되며, 세미콜론으로 끝난다.

<br>

'코드 주입 공격 - 예제' 에 대해 공부했다.

- 공격자는 SQL 명령어를 입력으로 전송하여 데이터베이스 기반 웹 사이트를 공격한다.
- 입력값에 세미콜론을 넣어, 서버에서 원래 명령 다음에 다른 명령을 실행하도록 한다.
- 버그를 악용하면, 체계에 직접 침입하지 않고도, 접근 권한이 없어도, 공격할 수 있다.
- 이는, 오늘날에는 거의 통하지 않는 방법이며, 이외에도 더 수준 높은 공격들도 있다.
- 프로그래머는 입력값의 잠재적 위험을 경계하고, 프로그램은 신중히 작성해야 한다.
- 아이디나 비밀번호에 특수 기호를 포함하지 않도록 규정함으로써 방어할 수도 있다.
- 우수한 서버의 경우, SQL 질의 실행 전에 특수 기호에 대한 표백 작업을 하기도 한다.
