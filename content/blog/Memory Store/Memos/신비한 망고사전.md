## 작성 규칙

```
- 완성에 목적을 두지 않고, 지식 축적에 목적을 둘 것
- 용어들의 뜻을 최대한 간략하게 정리하여 작성할 것
- 추가 내용은 별도의 글로 작성하여 링크 추가할 것
- 영어로 번역해도 의미가 변질되지 않도록 작성할 것
```
