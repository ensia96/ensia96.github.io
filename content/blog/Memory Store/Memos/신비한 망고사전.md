## ㄱ

- 골벵이표(`@`, At Sign, 앳)

## ㄴ

- 논리 회로(Logic Gate)
- 느낌표(`!`, Exclamation Mark) : 일반적으로 뱅(Bang), 스크리머(Screamer) 등으로 불림

## ㄷ

- 따옴표(`'` or `"`, Quotation Mark, Quotes) : 인용부, 인용 부호라고도 함

## ㄹ

- 리처드 브레이스웨이트(Richard Braithwait)

## ㅁ

- 물결표(`~`, Tilde, 틸다) : 홈 디렉토리(Unix 계열 OS), 부정(NOT) 연산자(C 언어) 를 나타냄
- 물음표(`?`, Question Mark)
- 밑줄 문자(`_`, Underscore, 언더스코어) : 'Underbar' 라고도 부름, 공백 문자 대신 사용되기도 함

## ㅂ

- 바이트(Byte)
- 백분율 기호(`%`, Percent Sign)
- 별표(`*`, Asterisk, 애스터리스크) : 'Star' 라고 발음함(전산학, 수학)
- 붙임표(`-`, Hyphen, 하이픈)
- 비트(Bit)
- 빗금(`/`, Slash, 슬래시) : 'Solidus' 라고도 부름(유니코드)

## ㅅ

- 삿갓표(`^`, Caret, 캐럿) : 비트 XOR 또는 지수 연산자로 사용됨(일부 프로그래밍 언어)
- 소프트웨어(Software)
- 수직선 기호(`|`, Pipe, 파이프) : 'Bar', 'Stick' 이라고도 부름, 논리합(OR) 연산자로 사용됨
- 쉼표(`,`, Comma, 콤마)
- 스네이크 케이스(Snake Case) : `this_is_snake_case`
- 쌍점(`:`, Colon, 콜론)
- 쌍반점(`;`, Semicolon, 세미콜론)

## ㅇ

- 앰퍼샌드(`&`, Ampersand) : 논리곱(AND) 연산자로 사용됨
- 역슬래시(`\`, Backslash, 백슬래시) : 'Escape' 라고 부름(C 언어, Unix)
- 우물정자(`#`, Hash, 해시) : '번호 기호', '샵', 'Number Sign', 'Pound Sign' 이라고도 부름
- 운영체제(Operating System, OS)
- 의사결정(Decision Making)
- 인터넷(Internet)

## ㅈ

- 중앙 처리 장치(Central Processing Unit, CPU)

## ㅊ

- 찰스 배비지(Charles Babbage)
- 추상화(Abstraction)

## ㅋ

- 카멜 케이스(Camel Case) : `thisIsCamelCase`
- 컴포넌트(Component) : 구성 요소
- 컴퓨터(Computer)
- 컴퓨팅(Computing)

## ㅌ

- 트랜지스터(Transistor)

## ㅍ

- 파스칼 케이스(Pascal Case) : `ThisIsPascalCase`
- 표준 스트림(Standard Strean)
- 표준 입력(Standard Input, stdin)
- 표준 출력(Standard Output, stdout)
- 프로그래밍(Programming)

## ㅎ

- 하드 디스크 드라이브(Hard Disk Drive, HDD)
- 하드웨어(Hardware)
