---

스크롤바의 선택자는 `::-webkit-scrollbar` 와 `::-webkit-scrollbar-thumb` 다.

1. '스타일을 변경하려는 스크롤바' 가 표시되는 HTML 태그 또는 React 컴포넌트를 찾는다.
2. CSS 에서, 선택자를 기준으로 해당 태그(컴포넌트) 의 스타일이 지정되는 영역을 찾는다.
3. 해당 영역 내부에 스크롤바 선택자를 적고, 뒤에 '{}' 를 붙여, 스타일 지정 영역을 만든다.
4. 마지막으로, 이렇게 만든 '스크롤바 스타일 지정 영역' 에 원하는 스타일을 추가하면 된다.
