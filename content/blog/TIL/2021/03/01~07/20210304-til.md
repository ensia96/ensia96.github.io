# 오늘 한 일

- Crash Course 내용 정리
   - '3-1. 조건 정리' 까지 정리했다.
- 'ALU'(산술 논리 장치) 에 대해 공부했다.
   - 'Arithmetic & Logic Unit' 을 줄인 표현이다.
   - 산술(Arithmetic) 장치와 논리(Logic) 장치로 구성되어 있다.
   - 컴퓨터의 모든 연산을 처리하는 핵심 장치이다.
   - 중앙 처리 장치(CPU) 의 구성요소다.
- '74181' 에 대해 공부했다.
   - 1970년에 출시된 '최초의 단일 칩 ALU' 이다.
   - 74181이 등장하기 전까진 하나의 칩에 온전한 ALU 를 집어넣지 못했다고 한다.
   - 74181 덕분에, 더 저렴하고 작고 성능 좋은 컴퓨터들이 등장할 수 있었다.
- ALU의 산술 장치에 대해 공부했다.
   - 덧셈, 뺄셈 등 컴퓨터의 모든 수치 연산을 담당한다.
   - 값을 하나씩 증가시키는 증분 연산 등의 다양한 단순 작업도 처리한다.
- 반가산기에 대해 공부했다.
   - 1비트 단위의 2진수에 대한 연산을 처리하는 회로다.
   - 입력 A, B를 연산하여, 합(sum)과 올림 값(carry)을 반환한다.
   - 합 출력을 담당하는 XOR 회로와 올림 값을 담당하는 AND 회로로 구성되어 있다.
- 전가산기의 구성 조건을 공부했다.
   - 반가산기는 1비트 단위의 계산까지 밖에 처리하지 못한다.
   - 더 큰 값을 계산하기 위해선, 올림 값까지 계산할 수 있어야 한다.
   - 결과 값은 반가산기와 마찬가지로 합과 올림 값으로 구성된다.
