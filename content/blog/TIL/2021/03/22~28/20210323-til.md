- 슈퍼스칼라에 대해 공부했다.
   - 여려 명령을 동시에 처리하기 위해 파이프라인의 수를 늘리는 기법이다.
      1. 명령을 수행할 때, 작업을 처리하는 부분 외에는 아무것도 하지 않는다.
      2. 한 번에 많은 작업을 수행하면 이런 부분들을 활용할 수 있게 된다.
      3. 많은 명령을 동시에 수행하도록 파이프라인의 수를 늘린다.
   - 여기에 추가로, 자주 사용되는 명령을 위해 같은 회로를 여러 개 사용할 수도 있다.
      - ALU가 2개 있으면, 동시에 2개의 수학 연산을 수행할 수 있게 되는 등
- 멀티 코어 프로세서에 대해 공부했다.
   - 명령 수행량 최적화와는 다른 방식이다.
      - '하나로 안되면 여러 개를 쓰면 되지!' 의 방식..
   - CPU의 명령 수행 단위(processing unit) 를 코어(core) 라고 한다.
   - 여러 개의 코어를 사용하기 때문에 멀티 코어라고 한다.
   - 독립된 각각의 코어가 작업을 처리하지만, CPU 내부의 자원을 공유한다.
   - 덕분에, 여러 코어가 작업을 나눠서 수행할 수도 있다.
- 슈퍼컴퓨터에 대해 공부했다.
   - 평범하지 않은 수준의 계산을 하기 위해 만들어졌다.  
     `(과학적인 이론의 시뮬레이션이라던가..)`
   - 하나의 컴퓨터에 여러 개의 CPU를 사용한다.  
     `(용도만큼이나 괴랄할 정도로 많이 사용된다..)`
   - 코어 별 성능, 코어의 수, 프로세서의 수가 대단히 많다.
- 플롭스라는 단위에 대해서 알게됐다.
   - 컴퓨터가 초당 수행할 수 있는 부동소수점 연산의 횟수다.
   - 클럭 속도가 아니라 실제 연산의 수행 속도다.
   - 컴퓨터의 성능을 수치로 표현할 때 사용된다.
