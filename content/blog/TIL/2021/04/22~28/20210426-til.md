# 오늘 한 일

- 자료 구조, 알고리즘 수업 정리
   - '순차적 자료구조: 배열과 리스트' 라는 강의 내용을 정리했다.
- 배열과 리스트에 대해 공부했다.
   - 배열은 연속적으로 값을 저장하는 형태의 순차적인 자료 구조다.
   - 아주 기본적인 자료 구조이지만, 가장 많이 사용되므로 그만큼 중요하다.
   - 파이썬에도 array 가 있지만, 리스트를 더 자주 사용한다.
   - 배열에서 원소의 위치를 나타내는 값을 인덱스라고 부른다.
   - 인덱스를 이용해 배열의 특정 위치에 있는 값을 상수 시간 내에 읽고 쓸 수 있다.
   - 이러한 기본 연산을 제공하는 자료 구조를 통틀어서 배열이라고 한다.
   - C 언어의 배열은 읽기와 쓰기 연산만 제공하지만, 파이썬의 리스트는 다양한 연산을 지원한다.
   - 배열이나 리스트에 저장할 수 있는 값의 개수를 용량이라고 한다.
   - 파이썬의 리스트는 용량을 자동으로 조절하는 동적 배열이다.
   - C 언어의 배열은 선언할 때 무조건 크기를 정해야 한다.
   - C 언어에서는 malloc(), calloc() 등의 할당 함수를 통해 용량을 직접 조정해야 한다.
   - 파이썬은 메모리가 부족하면 늘리고, 남으면 줄이는 것을 내부적으로 처리한다.
   - 파이썬의 리스트는 단순한 자료 구조가 아닌 클래스다.
      - 다양한 기능을 구현하기 위해 내부적으로 여러 정보를 관리한다.
