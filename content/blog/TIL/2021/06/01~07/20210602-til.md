      - 누락된 이미지를 추가하고, 제목 번호를 수정했다. ('d3af996')
- 점화식과 풀이법에 대해 공부했다.
   - 점화식은 식을 전개하여 푸는 방법 외에도 추측과 귀납법을 통해 풀 수 있다.
   - 추측하는 방법은 따로 정해져 있지 않으며, 경험에 의해 추측할 수 있다.
   - 귀납법은 어떠한 식이 성립한다는 것을 증명하는 데에 자주 사용되는 방법이다.
   - 이러한 귀납법은 추측한 내용에 대해서 증명하는 상황에서도 사용할 수 있다.
   - MoM 알고리즘의 점화식 T(n) 에 대해 T(n) <= 44n 이라고 추측할 수도 있다.
- 점화식을 귀납법을 이용해 증명해봤다.
   - 1 + 2 + 3 + … + n = (n(n + 1) / 2) 이라는 등식이 성립함을 증명했다.
   - 우선, n이 가장 작은 상황인 바닥 조건에서 주어진 식이 성립함을 확인한다.
      - n = 1 을 대입하면, 1 = (1(1 + 1) / 2) = 1 이므로, 등식이 성립한다.
   - 다음으로, n보다 작은 모든 경우에 대해, 주어진 등식이 성립한다고 가정한다.
   - 마지막으로, 아직 증명되지 않은 n에 대해, 주어진 등식이 성립함을 증명한다.
      - 바닥 조건이 성립하고, n보다 작은 모든 값에 대해 식이 성립한다고 가정했다.
      - 다시 말하면, n을 제외한 모든 값에 대해서, 주어진 식이 성립하는 상황이다.
      - 결국, n에 대해서도 식이 성립함을 보이면, 식의 성립이 연쇄적으로 증명된다.
   - (n - 1) 은 n보다 작은 값이기 때문에, (n - 1) 에 대해, 주어진 등식이 성립한다.
   - 따라서, 주어진 등식을 변환한 1 + 2 + … + (n - 1) = (n(n - 1) / 2) 도 성립한다.
   - 여기에 n을 더하고 식을 정리하면, (n(n + 1) / 2) 이 되면서, 증명이 마무리된다.
   - 귀납법은 이렇게 바닥 조건부터 하나씩 연쇄적으로 식이 성립함을 증명하는 방법이다.
- MoM 알고리즘의 수행 시간에 대해 공부했다.
   - MoM 알고리즘의 수행 시간을 나타내는 T(n) 에 대한 점화식을 귀납법을 이용해 풀어봤다.
   - 우선, T(n) <= 44n 이 성립한다고 추측했고, 이 부등식이 바닥 조건에서 성립함을 증명했다.
      - T(1) = 1 이기 때문에, n에 1을 대입했을 때의 T(1) <= 44 이 성립하게 된다.
   - 다음으로, n보다 작은 모든 값에 대해, 주어진 부등식 T(n) <= 44n 이 성립한다고 가정했다.
   - 마지막으로, n인 경우에 대해 성립함을 증명하기 위해, 기존의 점화식도 함께 풀었다.
      - 이 때, MoM 알고리즘의 점화식은 T(n) = T(3n / 4) + T(n / 5) + (11n / 5) 다.
   - (3n / 4) 와 (n / 5) 는 무조건 n보다 작은 수이므로, T(n) <= 44n 이 성립한다.
   - 따라서, T(3n / 4) <= 44 * (3 / 4), T(n / 5) <= 44 * (n / 5) 라고 할 수 있다.
   - 이를 기존의 점화식에 적용하면, 33n + (44n / 5) + (11n / 5) = 44n 이 된다.
   - 결국, 식을 풀면, T(n) = 44n <= 44n 가 되므로, T(n) <= 44n 이라는 것이 증명된다.
   - T(n) <= 44n 이므로, MoM 알고리즘의 수행에 필요한 수행 시간은 상수 시간이 된다.
   - 이 때, 44n은 임의의 상수 c에 대한 점화식 T(n) <= (c * n) 에 귀납법을 적용해 얻은 값이다.
