# 오늘 한 일

- 자료 구조, 알고리즘 수업 정리
   - '알고리즘 - 분할정복 - 큰수 곱셈 (Karatsuba 알고리즘)' 라는 강의 내용을 정리했다.
- 카라추바 알고리즘의 분할 아이디어에 대해 공부했다.
   - 카라추바 알고리즘은 n자리 숫자 2개, u와 v를 입력으로 받아서 빠르게 곱하는 알고리즘이다.
   - 우선, 입력으로 받은 u와 v를, 자리수를 기준으로 반씩 나눈 후에, 각각을 x, y, w, z라고 한다.
   - 즉, u의 1의 자리를 0번째라고 했을 때, x는 (n - 1) 번째부터 (n / 2) 번째까지의 숫자가 된다.
   - 그리고, x의 실제 크기를 표현하기 위해서는, x에 나머지 절반만큼의 자리수를 추가해야 한다.
   - 그러므로, x의 실제 크기는 x * 10^(n / 2) 이며, u = (x * 10^(n / 2)) + y 라고 할 수 있다.
   - 또 다른 입력 v에도, 이 과정을 그대로 반복하면, v = (w * 10^(n / 2)) + z 라고 할 수 있다.
   - 이를 적용하면, u * v = (x * 10^(n / 2) + y) * ((w * 10^(n / 2)) + z) 라고 표현할 수 있다.
   - 이를, 풀어서 다시 정리하면, u * v = (xw * 10^n) + ((xz + yw) * 10^(n / 2)) + yz 가 된다.
   - 등식을 보면 알 수 있듯, n자리 숫자 u와 v를 곱하려면, xw, xz, yw, yz를 모두 계산해야 한다.
   - 이 때, xw, xz, yw, yz는, n의 자리 숫자를 반으로 나눈, (n / 2) 자리 숫자 2개를 곱한 것이다.
   - 그리고, 이렇게 (n / 2) 자리 숫자 2개를 곱하면, 그것이 가질 수 있는 자리수의 최대값은 n이다.
   - 이 값들을 구한 다음에는, 자리수를 맞추기 위한 곱셈과 모든 항에 대한 덧셈을 수행해야 한다.
   - 결국, n자리 숫자 2개를 곱하는 문제가, 절반 크기의 문제 4개와 추가 기본 연산으로 나뉜다.
   - 이 때, 문제 해결에 필요한 추가 기본 연산의 횟수는, xw, xz, yw, yz의 자리수인 n에 비례한다.
   - 그러므로, 임의의 상수를 c라고 했을 때, 필요한 추가 기본 연산의 횟수는 cn이라 할 수 있다.
   - 따라서, 이 방식의 기본 연산 횟수를 T(n) 이라고 하면, T(n) = (4 * T(n / 2)) + cn 이 된다.
   - T(1) = c, n = 2^k 이라고 가정하고, 이 점화식을 풀어보면, 결국, T(n) = O(n^2) 이 된다.
   - 이러한 풀이 방식은, 분할 정복법을 적용했음에도, 기본적인 곱셈법보다 효율적이지 않다.
   - 따라서, 큰 수 곱셈 문제를 제대로 풀기 위해서는, 풀이 방식을 여기서 약간 바꿔줘야 한다.
