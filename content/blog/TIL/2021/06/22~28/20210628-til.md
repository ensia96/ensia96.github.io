# 오늘 한 일

- Crash Course 내용 정리
   - '6. 월드 와이드 웹의 발전' 까지 정리했다.
- 월드 와이드 웹의 등장에 대해 공부했다.
   - 최초의 웹 브라우저와 웹 서버는 1990년에 팀 버너스리 경에 의해 작성되었다.
   - 또, 이러한 과정에서, URL, HTML, HTTP와 같은 기본 웹 표준도 함께 만들어졌다.
   - 누구나 웹 서버와 웹 브라우저를 개발할 수 있도록 개방형 표준으로 공개되었다.
- 월드 와이드 웹의 발전에 대해 공부했다.
   - 문자와 그래픽을 한 화면에 표시할 수 있는 최초의 브라우저, 모자이크가 등장했다.
   - 대중들에게 기술 표준이 공개되고, 다양한 브라우저, 서버, 사이트가 개발되었다.
