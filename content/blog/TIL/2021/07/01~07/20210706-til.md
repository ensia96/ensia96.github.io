## 오늘 한 일

새로운 Pull Request 를 생성하고, 정리해둔 순서대로 카테고리 기능 수정 작업을 진행했다.

- slug 항목을 사용할 수 있도록 쿼리를 수정하고, 원하는 형태의 객체가 되도록 코딩했다.
- 내일의 목표는, '오늘 만든 객체를 재귀적으로 렌더링하는 컴포넌트' 를 구성하는 것이다.

회의를 통해 해커톤 프로젝트에 관한 의견을 교환했고, 목표와 작업 명세를 구체화했다.

- 만들기로 정한 서비스에 대해, 구체적인 부분까지 정하지 않아서 혼란이 있는 모양이다.
- 서비스 흐름은 다 같이 공감했으니, 중요도 순으로 세부 사항을 구체화하자고 제안했다.
- 전체 회의를 마치고 난 후, 백엔드 동료분과 모델링과 서비스 기능 구성 등을 상의했다.
- 회원 정보에 관련된 구성을 간단하게 정리했고, 대략적인 일정과 중간 목표를 설정했다.
- 그리고, 백엔드 동료분 덕분에, 'Diagram' 이라는 도표 정리 웹 서비스를 알게 되었다.
